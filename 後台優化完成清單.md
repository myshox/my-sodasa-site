## 📝 使用說明

### **操作日誌查詢**
在 Supabase Dashboard 中執行：
```sql
-- 查看最近 50 筆操作
SELECT * FROM audit_logs ORDER BY created_at DESC LIMIT 50;

-- 查看特定管理員的操作
SELECT * FROM audit_logs WHERE admin_username = 'admin' ORDER BY created_at DESC;

-- 查看今天的操作
SELECT * FROM audit_logs WHERE created_at >= CURRENT_DATE ORDER BY created_at DESC;
```

### **CSV 匯出**
1. 進入「贊助管理」分頁
2. 使用搜尋/篩選功能選擇要匯出的資料
3. 點擊右上角「匯出 CSV」按鈕
4. 檔案會自動下載

### **分頁使用**
1. 使用右下角的分頁控制
2. 可選擇每頁顯示 10/20/50/100 筆
3. 支援快速跳頁（第一頁、最後一頁）
