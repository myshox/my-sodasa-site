## 1️⃣ 圖表視覺化系統 ✅

### **功能概述**
在「報表統計」分頁加入三種互動式圖表，讓管理員能直觀了解贊助趨勢和分布。

### **實作內容**

#### **📈 贊助趨勢折線圖**
- **數據範圍**：最近 30 天
- **顯示內容**：每日贊助總金額
- **特色**：
  - 平滑曲線 (tension: 0.4)
  - 漸層填充背景
  - Hover 顯示詳細金額
  - 金色配色主題
  - Y 軸自動格式化 ($1k, $2k...)

#### **🥧 支付方式圓餅圖**
- **顯示內容**：各支付方式的金額占比
- **支援方式**：SHOPLINE、LINE Pay 等
- **特色**：
  - 甜甜圈圖表 (doughnut)
  - 多彩配色（藍、綠、紅、金）
  - Hover 放大效果
  - 圖例顯示在底部

#### **📊 熱門方案柱狀圖**
- **顯示內容**：TOP 10 最受歡迎方案
- **數據**：訂單數量統計
- **特色**：
  - 圓角柱狀圖
  - 彩虹配色系統
  - 排序顯示（由高到低）
  - 顯示訂單數量

### **技術實作**
- 使用 **Chart.js 4.4.1**
- React `useRef` + `useEffect` 管理圖表生命週期
- 自動銷毀舊圖表防止記憶體洩漏
- 響應式設計（`maintainAspectRatio: false`）

### **UI 設計**
- 深色主題配色
- 圖表卡片設計（石紋背景 + 邊框）
- 圖示標題（TrendingUp, CreditCard, BarChart3）
- 無數據時顯示友善提示

## 2️⃣ 標籤系統 ✅

### **功能概述**
為贊助紀錄加入標籤分類功能，方便管理員快速識別和篩選不同類型的贊助者。

### **資料庫設計**

#### **SQL Schema** (`add_tags_to_donations.sql`)
```sql
-- 標籤欄位（PostgreSQL 陣列）
ALTER TABLE donations ADD COLUMN tags TEXT[] DEFAULT '{}';

-- GIN 索引優化查詢
CREATE INDEX idx_donations_tags ON donations USING GIN(tags);
```

#### **自動標籤規則**
- ✅ **VIP**：單筆贊助 >= NT$ 5,000（自動）
- ✅ **常客**：累計贊助 >= 3 次（自動）
- 🏷️ **大戶**：累計贊助 >= NT$ 10,000（手動）
- 🌱 **新手**：首次贊助（手動）
- 📅 **月訂**：每月固定贊助（手動）

#### **PostgreSQL 觸發器**
- `auto_tag_vip()`：自動加上 VIP 標籤
- `check_and_tag_regular()`：自動檢查並標記常客

### **前端功能**

#### **標籤顯示**
- 表格中獨立一列顯示標籤
- 彩色標籤徽章（依類型不同顏色）
- 每個標籤可點擊「X」移除

#### **標籤管理**
- ➕ 點擊「+」按鈕展開標籤編輯
- 快速加入預設標籤（一鍵操作）
- 自訂標籤顏色配置

#### **標籤操作**
- `addTag()`：為單筆紀錄加入標籤
- `removeTag()`：移除標籤
- `batchAddTag()`：批次加入標籤
- 所有操作記錄到 `audit_logs`

### **標籤配色**
```javascript
VIP      → 金色 (gold-400)
常客     → 風屬性綠 (wind-400)
大戶     → 火屬性紅 (fire-400)
新手     → 水屬性藍 (water-400)
月訂     → 紫色 (purple-400)
```

## 3️⃣ 進階搜尋系統 ✅

### **功能概述**
提供多維度的資料篩選功能，讓管理員能快速找到目標贊助紀錄。

### **搜尋條件**

#### **📅 日期範圍**
- 開始日期 (dateFrom)
- 結束日期 (dateTo)
- 自動包含當天 23:59:59

#### **💰 金額範圍**
- 最低金額 (amountMin)
- 最高金額 (amountMax)
- 支援無上限查詢

#### **💳 支付方式**
- 全部
- SHOPLINE
- LINE Pay

#### **🏷️ 標籤篩選**
- 全部
- VIP
- 常客
- 大戶
- 新手
- 月訂

### **UI 設計**

#### **摺疊式介面**
- 點擊「進階搜尋」按鈕展開/摺疊
- 按鈕高亮顯示（風屬性綠色）
- 齒輪圖示 (Settings icon)

#### **搜尋面板**
- 深色卡片設計
- 4 欄網格布局（響應式）
- 每個欄位有清楚標籤
- 「清除篩選」按鈕重置所有條件

### **搜尋邏輯**
```javascript
// 多條件 AND 邏輯
matchSearch && matchStatus && matchDate && matchAmount && matchPayment && matchTag
```

### **特色功能**
- ✅ 即時篩選（輸入即生效）
- ✅ 自動重置分頁到第一頁
- ✅ 顯示篩選結果數量
- ✅ 搭配基本搜尋使用

## 🚀 使用指南

### **步驟 1：執行 SQL**
在 Supabase Dashboard → SQL Editor 執行：
```sql
-- 1. 操作日誌表（如已執行可跳過）
執行 create_audit_logs.sql

-- 2. 標籤系統
執行 add_tags_to_donations.sql
```

### **步驟 2：測試功能**

#### **圖表視覺化**
1. 登入後台
2. 切換到「報表統計」分頁
3. 查看三種圖表：
   - 贊助趨勢（折線圖）
   - 支付方式分布（圓餅圖）
   - 熱門方案（柱狀圖）

#### **標籤系統**
1. 進入「贊助管理」分頁
2. 查看表格中的「標籤」列
3. 點擊「+」加入標籤
4. 點擊標籤上的「X」移除標籤
5. 使用進階搜尋依標籤篩選

#### **進階搜尋**
1. 點擊「進階搜尋」按鈕
2. 設定日期範圍、金額範圍等條件
3. 查看篩選結果
4. 點擊「清除篩選」重置

#### **批次操作**
1. 勾選多筆紀錄的 checkbox
2. 點擊右上角「批次操作 (X)」
3. 選擇要執行的操作：
   - 批次更新狀態
   - 批次加入標籤
   - 批次刪除
4. 確認操作
