# 贊助管理手機版卡片設計

## 更新內容

重新設計贊助管理頁面的手機版布局，從複雜的表格改為精美的卡片式設計，大幅提升手機端的可讀性和操作體驗。

## 設計理念

### Before（表格）
❌ 需要橫向滾動  
❌ 資訊密集難以閱讀  
❌ 觸控按鈕太小  
❌ 欄位被壓縮  

### After（卡片）
✅ 垂直滾動瀏覽  
✅ 資訊分層清晰  
✅ 大觸控區域  
✅ 完整顯示所有資訊

## 詳細設計

### 1. 卡片頭部
**功能**：狀態和時間一目了然

**佈局**：
- 左側：選擇框 + 狀態標籤
- 右側：日期（簡化顯示）

**樣式**：
```jsx
bg-stone-800/50 px-4 py-3
```

**狀態標籤**：
- ✅ 已完成：綠色
- ⏳ 待處理：黃色
- ❌ 已取消：紅色

### 2. 玩家資訊區塊
**功能**：顯示玩家身份資訊

**包含**：
- 👤 圖標標題
- 遊戲帳號（大字體、粗體）
- Email（小字體、可換行）
- LINE 名稱（綠色高亮）

**樣式**：
```jsx
space-y-1
text-white font-bold text-sm  // 帳號
text-stone-400 text-xs       // Email
text-green-400               // LINE
```

### 3. 金額區塊
**功能**：突出顯示金額和金幣

**設計**：
- 深色背景區塊（bg-stone-800/50）
- 兩行佈局
  - 第一行：方案名稱標籤
  - 第二行：金額（左）+ 金幣（右）

**金額顯示**：
```jsx
text-gold-400 font-black text-xl  // 非常醒目
```

**金幣顯示**：
```jsx
bg-gold-500/10 px-2 py-1 rounded-lg  // 帶背景
flex items-center gap-1              // 圖標+數字
```

### 4. 支付資訊
**功能**：簡潔顯示支付方式和IP

**佈局**：橫向排列，間距分隔

**樣式**：
```jsx
flex items-center gap-4 text-xs
```

**圖標**：
- 💳 CreditCard for 支付方式
- IP code block 顯示

### 5. 標籤區域
**功能**：顯示分類標籤

**樣式**：
- flex-wrap 自動換行
- 各標籤有不同顏色主題
- 小字體（text-xs）

### 6. 備註區域
**設計**：
- 淺灰背景
- 左側藍色邊框條
- 圓角設計

**樣式**：
```jsx
bg-stone-800/30 rounded-lg p-2
border-l-2 border-wind-500
```

### 7. 操作按鈕
**功能**：編輯和刪除操作

**佈局**：
- 底部固定區塊
- 編輯（佔大部分寬度）+ 刪除

**樣式**：
```jsx
// 編輯按鈕
bg-wind-600/20 text-wind-400 border-wind-600/30

// 刪除按鈕
bg-red-600/20 text-red-400 border-red-600/30
```

**觸控優化**：
- py-2 提供足夠觸控高度
- gap-2 按鈕間距
- 圖標+文字清晰

## 響應式策略

### 桌面版（≥ lg）
```jsx
className="hidden lg:block"  // 顯示表格
```
- 保留完整表格
- 所有欄位可見
- 適合大量資料瀏覽

### 手機版（< lg）
```jsx
className="lg:hidden"  // 顯示卡片
```
- 卡片式佈局
- 垂直滾動
- 更大觸控區域

## 技術實現

### 卡片容器
```jsx
<div className="lg:hidden p-4 space-y-4">
    {paginatedData.map((item) => (
        // 卡片內容
    ))}
</div>
```

### 狀態樣式映射
```jsx
{item.status === 'completed' ? 
    'bg-green-500/20 text-green-400 border border-green-500/50' :
 item.status === 'pending' ? 
    'bg-yellow-500/20 text-yellow-400 border border-yellow-500/50' :
    'bg-red-500/20 text-red-400 border border-red-500/50'
}
```

### 金幣顯示邏輯
```jsx
{item.coins?.toLocaleString() || 
 (PLANS.find(p => p.total === item.amount)?.total || 0).toLocaleString()}
```

## 優化細節

### 1. 文字斷行
```jsx
className="break-all"  // Email 可完整顯示
```

### 2. 間距系統
- 卡片間距：space-y-4
- 卡片內邊距：p-4
- 區塊間距：space-y-3
- 行內間距：gap-1, gap-2

### 3. 顏色系統
- 背景：stone-900, stone-800
- 邊框：stone-700
- 文字：white, stone-400, stone-300
- 強調：gold-400 (金額)
- 狀態：green/yellow/red

### 4. 圓角設計
- 卡片外層：rounded-2xl
- 內部區塊：rounded-xl, rounded-lg
- 標籤：rounded-full

### 5. 圖標尺寸
- 標題圖標：14px
- 支付圖標：12px
- 按鈕圖標：14px

## 手機版體驗提升

### Before vs After
| 項目 | Before | After |
|------|--------|-------|
| 滾動方向 | 橫向 | 垂直 |
| 資訊密度 | 極高 | 適中 |
| 觸控面積 | 小 | 大 |
| 可讀性 | 低 | 高 |
| 操作便利性 | 難 | 易 |

### 用戶操作流程
1. 📱 **垂直滾動**瀏覽所有贊助
2. 👀 **快速識別**狀態（顏色標籤）
3. 📊 **查看詳情**（卡片內完整資訊）
4. ✏️ **點擊編輯**或刪除（大按鈕）

## 注意事項

- 卡片和表格使用相同的 `paginatedData`
- 分頁控制對兩種佈局都有效
- 搜尋和篩選功能通用
- 批次操作在兩種佈局都可用
