# 活動彈窗主打功能

## 功能說明

1. **後台可編輯**：在後台「活動管理」中，每則活動旁可設為「彈窗主打」或「取消主打」
2. **僅一個主打**：僅能有一則活動為主打，設定新的會自動取消舊的
3. **彈窗圖片比例**：新增/編輯活動可選 1:1、16:9、9:16
4. **顯示圖片**：活動內容中的圖片會自動擷取第一張顯示於彈窗

## 執行步驟

1.  Supabase Dashboard → SQL Editor
2.  執行 `040_events_popup_featured.sql`
3.  執行 `041_events_popup_aspect.sql`

## 後台操作

- 進入後台 → 活動管理
- 在任一活動旁點擊 ⭐ 設為彈窗主打
- 已為主打的活動可點「取消主打」解除
- 編輯活動時可選擇「彈窗圖片比例」：1:1 正方、16:9 橫幅、9:16 直幅

## 前台效果

- **右下角彈窗**：主打活動僅一個，排第一、有金色標籤，圖片依設定比例顯示
- **其他活動**：無星號，圖片依各自比例顯示
