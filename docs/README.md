## 🔧 **技術決策記錄**

### **為什麼選擇 Supabase？**
- ✅ 完整的 BaaS 解決方案
- ✅ 內建認證系統（bcrypt）
- ✅ Row Level Security
- ✅ Real-time 功能
- ✅ 免費額度充足

### **為什麼使用單頁應用（SPA）？**
- ✅ 簡化部署（只需 GitHub Pages）
- ✅ 無需後端伺服器
- ✅ 快速載入
- ✅ 適合小型專案

### **為什麼選擇 Chart.js？**
- ✅ 輕量級（~200KB）
- ✅ 簡單易用
- ✅ 響應式設計
- ✅ 完整的圖表類型

## 🆘 **常見問題**

### **Q: 如何新增管理員？**
A: 在後台的「超管」分頁，點擊「新增管理員」

### **Q: 忘記管理員密碼？**
A: 使用 Supabase Dashboard 的密碼重設功能

### **Q: 如何備份資料庫？**
A: Supabase Dashboard → Database → Backups

### **Q: 如何查看審計日誌？**
A: 後台管理 → 執行 SQL 查詢或查看 audit_logs 表
